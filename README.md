# Win Neonlicht-Service

Website für Win Neonlicht-Service – Neonlicht, Werbetechnik und Gebäudereinigung.

## Technologien

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Lokale Entwicklung

```sh
npm install
npm run dev
```

Die App läuft standardmäßig unter `http://localhost:8080/`.

## Build

```sh
npm run build
npm run preview
```
